# 💎 零壹启动器
## ❓ 什么是零壹启动器？
零壹启动器（简称“ZOMCL”）是一个跨平台 Minecraft Bedrock 启动器。由[YOUQI4461](https://github.com/YOUQI4461/)维护。
## 🌹 如何使用？
打开浏览器，选择一个进入：
- [zomc.ynet0.dpdns.org](https://zomc.ynet0.dpdns.org/)
## 💻 如何自己部署？
把项目 Fork 之后打开 Github Pages 即可！
